Помощь

/help all

Показывает все доступные команды.

/help main

Показывает основные команды.

/help moderation

Показывает команды модерации.

/help moderation

Показывает ролевые команды.

/help moderation

Показывает команды для настроек.

/help join_members

Показывает помощь для настройки приветствий.

Настройки

/set color [#hex]

Задаёт цвет сообщений.

/set emoji [emoji]

Задаёт эмодзи.

/set max_warns [count]*

Задаёт максимальное количество предупреждений для бана.

/stats_setup [everyone_name]* [piples_name]* [bots_name]*

Создаёт каналы для отслеживания статистики.

/set join_channel [#channel]*

Задаёт канал для приветствий.

/set join_message [message]*

Задаёт приветственное сообщение.

/set join_image [image_url]*

Задаёт аватар при входе участников без аватарки.

Основные

/stats show

Показывает стаистику сервера.

/stats update

Обновляет каналы статистики.

/server

Показывает настройки бота на сервере.

/info [member]

Показывает информацию об учатснике.

/avatar [member]

Отправляет аватар пользователя.

/emoji [member]*

Отправляет аватар пользователя.

/rand [min] [max]

Отправляет случайное число.

/ball [question]*

Спрашивает у шара.

/math [excpression]*

Решает математическое выражение.

/like [member]*

Лайкает пользователя.

/dislike [member]*

Дизлайкает пользователя.

/reputation [member]*

Показывает репутацию пользователя.

/giveaway create [title]* [description]* [winners]* [reward]*

Создаёт розыгрыш.

Модерация

/clear [count]*

Чистит последние сообщения.

/nickname set [member]* [member]*

Устанавливает никнейм.

/nickname add [member]* [member]*

Добавляет символы к никнейму.

/nickname show [member]*

Показывает полный никнейм на сервере.

/mute [member]*[duration]

Мьютит участника.

/unmute [member]*

Размьючивает участника.

/mutes list

Показывает список мьютов.

/mutes clear

Чистит список мьютов.

/timeout [member]*

Выдаёт таймаут.

/untimeout [member]*

Снимает таймаут.

/warn [member]*

Выдаёт предупреждение.

/unwarn [member]*

Снимает предупреждение.

/warns show [member]*

Показывает предупреждения.

/warns list

Показывает список предупреждений.

/warns clear

Очистка предупреждений.

/kick [member]*

Кикает участника.

/ban [member]*

Банит участника.

/unban [member]*

Разбанивает участника.

/bans list

Показывает список банов сервера.

/bans clear

Чистит список банов сервера.

Роли

/roles list

Показывает список ролей сервера.

/roles add [member]*[role]*

Добавляет роль пользователю.

/roles remove [member]*[role]*

Забирает роль у пользователя.

/roles mass_add [role]*

Добавляет роль всем участникам сервера.

/roles mass_remove [role]*

Забирает роль у всех участников сервера.

Автороли

/autorole add [role]*

Добавляет роль в список авторолей.

/autorole remove [role]*

Убирает роль из списка авторолей.

/autorole list

Показывает список авторолей.

/autorole clear

Чистит список авторолей.

Автомодерация

/automod words [True/False]*

Включает автомодерацию плохих слов.

/automod links [True/False]*

Включает автомодерацию ссылок.

/automod invites [True/False]*

Включает автомодерацию приглашений.

/links_exceptions

Показывает исключения (ссылки).

/add_links_exception [link]*

Добавляет ссылку в исключения.

/remove_links_exception [link]*

Убирает ссылку из исключений.

/clear_links_exceptions

Чистит список исключений (ссылки).

/invites_exceptions

Показывает исключения (приглашения).

/add_invites_exception [invite_link]*

Добавляет приглашение в исключения.

/remove_invites_exception [invite_link]*

Убирает приглашение из исключений.

/clear_invites_exceptions

Чистит список исключений (приглашения).
